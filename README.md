# muhammed-iqbal.dev

Personal portfolio site for Muhammed Iqbal — AI Application Engineer based in Singapore.

**Live:** [muhammed-iqbal.dev](https://muhammed-iqbal.dev)

## Stack

- Next.js 16 (App Router)
- Tailwind CSS v4
- Deployed on Vercel

## Projects featured

1. **Auto Negotiator** — Multi-tool AI agent automating procurement negotiation
2. **Procurement Classifier** — ML + Flask single-click categorization tool
3. **Spatial Wayfinding System** — PDF → SVG → 3D web rendering pipeline
4. **EUCD Device Kiosks** — Hardware-software systems integration for device lifecycle
5. **Zebra Printer Fleet Tool** — Global fleet management dashboard with headless ZPL printing
6. **Tourism Photobooth** — Generative AI photobooth experience

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
